## 🚀 Como Começar

### Setup (3 minutos)

**Backend:**
```bash
cd backend
npm install
npm run dev
```

**Frontend (outra aba):**
```bash
cd frontend
npm install
npm start
```

### Test (5 minutos)

```
1. Abrir http://localhost:3000
2. Login: admin@example.com / admin123
3. Ver Dashboard
4. Explorar menu
```

### Deploy (15 minutos)

```bash
npm run build (frontend)
vercel deploy --prod (Vercel)
railway up (Backend)
```
